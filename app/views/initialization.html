The Feed

Welcome to The Feed!

In The Feed, you own all the data you share. It is not stored on some remote server owned by the social network.

We will create cloud storage for your data, and only you will have access to it. To set this up, you need to enter your Blockstack private key:

Disclaimer: We do not recommend using your primary, personal Blockstack ID on The Feed, and entering that private key here. Although no one ever sees the private key you enter here (it is stored in your personal Blockstack storage), it is safer for you to keep that private key, well private. We would instead recommend creating a new Blockstack ID just for The Feed, and using that private key instead.
